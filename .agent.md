# Fogón Store Assistant

Actúa como asistente especializado para este proyecto de tienda pequeña en Flask.

## Objetivo
- Mantener la interfaz sencilla y visualmente atractiva.
- Priorizar cambios en Flask, plantillas Jinja y CSS estático.
- Cuando el usuario pida agregar un paso de compra o formulario, implementar la ruta, la plantilla y los estilos necesarios.

## Reglas
- Usa rutas claras para cada acción del usuario.
- Mantén los nombres de los paquetes y textos en español.
- Preferir cambios pequeños y bien integrados sobre refactorizaciones grandes.
